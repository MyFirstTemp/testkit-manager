Testkit Overview

Testkit is one testing execution framework, which is composed by Testkit-Manager and Testkit-Lite.

  • Testkit-Manager represents a GUI front-end in browser. Testkit-Manager manages the auto test cases execution remotely, and it provides unified web UI to help manual tests execution.
  • Testkit-Lite is the default core test runner driving test execution in Testkit-Manager. It also provides more comprehensive options for your testing in command line mode.

Testkit Quick Start

1.Download and install Testkit-Manager to host machine

(1) Install dependency packages and testkit-manager package:

a. If Host Machine is preinstalled Ubuntu (take ubuntu11.10 as an example), then

        sudo apt-get install libxml-simple-perl

        sudo apt-get install libdigest-sha-perl

        sudo apt-get install libjson-perl

        sudo apt-get install libio-pty-easy-perl

        sudo apt-get install cpanminus

        cpanm JSON

        sudo apt-get install rpm

        sudo rpm -ivh testkit-manager-x.x.x-x.noarch.rpm

b. If host machine is preinstalled Fedora (take fedora17 as an example)

        sudo yum install perl-libxml-perl

        sudo yum install perl-App-cpanminus

        sudo yum install perl-XML-Simple

        sudo rpm -ivh testkit-manager-x.x.x-x.noarch.rpm --nodeps

(2) Install sdb tool on host machine. The sdb tool can be got from Tizen SDK.

    sudo cp <somewhere>/sdb /usr/bin

    sudo chmod 755 /usr/bin/sdb

2.Ensure the Target Device is connected to host machine

    power on device

    connect USB

    use 'sdb devices' from the host to check if device is connected successfully

3.Launch Testkit-Manager

    sudo /opt/testkit/manager/webapps/bin/testkit_start.pl

    Open web UI of Testkit-Manager in browser: http://<test-machine-name>:8899

4.Custom Test Plan

On the page for customizing test plan, you can choose test packages. The chosen test packages and Testkit-Lite packages will be automatically downloaded and installed to target device

5.Once all the tests are executed, you can view the test results in your browser. You can continue to run the tests in Testkit-Manager.

Only use Testkit-Lite

You can use Testkit-Lite directly without Testkit-Manager.

Please see details in README of testkit-lite.
